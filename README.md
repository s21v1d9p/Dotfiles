### **Repository Description**
This repository hosts my personal **Linux configuration dotfiles**, meticulously designed for a **highly efficient and minimalist** workflow. Key features include:

- **Zsh** configuration with `powerlevel10k` theme for a fast and informative terminal experience.
- **Neovim** setup for a productive and customized coding environment.
- **Dunst** notification daemon configuration for streamlined system alerts.
- **Kitty** and **Alacritty** terminal emulator configurations for performance and aesthetics.
- **Custom scripts** for monitoring system resources (battery, CPU, memory, network) and automating tasks.
- **Optimized dotfiles** for integration with:
  - **dwm (Dynamic Window Manager)**
  - **GTK-based applications**
  - **LightDM display manager**
